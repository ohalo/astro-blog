如果说多因子选股是量化世界的"基本功"，那统计套利就是量化世界的"进阶课程"。

统计套利的底层逻辑很优雅：找到两只（或多只）价格走势高度相关的股票，当它们的价差偏离正常范围时下注它会回归。你不需要判断市场涨跌，甚至不需要判断个股涨跌——你只需要判断它们之间的关系什么时候会"恢复正常"。

这听起来很有吸引力，但魔鬼在细节里。

![配对交易示意](/images/2026-06-01-stat-arbitrage-pairs-trading/pairs-trading.jpg)

## 配对交易的核心假设

配对交易建立在三个关键假设之上：

**假设一：存在稳定的长期均衡关系。** 两只股票的价差在长期维持在一个均值附近波动，偏离是暂时的。

**假设二：市场会修正错误定价。** 当价差偏离过大时，市场力量会把它推回正常水平。你可以赚这笔"回归"的钱。

**假设三：你可以识别真正的偏离。** 不是所有的价差扩大都是交易机会——有些可能是基本面发生了永久性变化，价差永远不会回去了。

前两个假设是配对交易的理论基础，第三个假设是区分"策略"和"赌博"的关键。我见过太多人把随机噪声当信号，买入一支持续下跌的股票然后等待一个永远不会来的"回归"。

## 相关性 ≠ 协整

这是统计套利中最常见的误解。

**相关性**衡量的是两只股票价格变动的同步程度——它们是不是一起涨一起跌。但相关性高不等于有稳定的价差关系。

举个例子：茅台和五粮液的日收益率相关性可能很高（都受白酒行业影响），但它们的价格比例可能在几年内从2:1变成3:1甚至4:1。股价可以"同步变动"但"渐行渐远"。

**协整**衡量的是两个序列之间是否存在长期均衡关系。如果两只股票的价格是协整的，那么它们的线性组合是平稳的——价差会在一个均值附近波动，不会无限发散。

用数学表达：如果存在一个 β 使得 `Spread = StockA - β × StockB` 是一个平稳序列（均值和方差不随时间变化），那 StockA 和 StockB 就是协整的。

判断协整的标准工具是 **ADF检验**（Augmented Dickey-Fuller test）：

```python
import numpy as np
import pandas as pd
from statsmodels.tsa.stattools import adfuller

def test_cointegration(stock_a, stock_b):
    """用ADF检验判断两只股票是否协整"""
    # 估计对冲比率（用OLS回归）
    hedge_ratio = np.polyfit(stock_b, stock_a, 1)[0]
    
    # 计算价差序列
    spread = stock_a - hedge_ratio * stock_b
    
    # ADF检验
    adf_result = adfuller(spread)
    p_value = adf_result[1]
    
    return {
        'hedge_ratio': hedge_ratio,
        'p_value': p_value,
        'is_cointegrated': p_value < 0.05
    }
```

p值小于0.05意味着价差是平稳的——有95%的把握认为它们之间存在协整关系。

## 实战中的四大陷阱

**陷阱一：样本内过拟合**

你用2019-2023年的数据找到一对"协整"的股票，然后在2024年实盘——发现价差再也没有回归过。为什么？因为你的样本期包含了新冠疫情的特殊影响，那段时间银行股和地产股的价差暂时缩小，但疫情后的结构变化使这个关系永久性断裂了。

解决方法：用滚动窗口回测，至少覆盖一个完整的牛熊周期。如果策略在多个子区间都表现稳定，你才有理由相信它不是在记忆历史。

**陷阱二：交易成本吃掉利润**

配对交易通常利润微薄。一个典型的配对交易可能只有2-3%的价差空间，而进出一次的双边交易成本（佣金+滑点+印花税）轻松吃掉0.3-0.5%。如果你的胜率只有60%，那净收益会非常薄。

更隐蔽的成本是融资成本。做空需要借券，年化成本可能到8%甚至更高。你的策略年化收益如果只有15%，扣掉借券成本后只剩7%——再扣掉滑点，可能就不如拿着指数基金了。

**陷阱三：价差"正常化"的时间不确定**

你发现了5个标准差的偏离，按正态分布这应该1个月之内回归。但市场可以持续不理性——价差可能从5σ变成8σ，而你的止损设在10σ。

这就是为什么配对交易必须有严格的止损纪律。你不能无限期等待均值回归，因为"均值"本身可能已经变了。

**陷阱四：流动性风险**

很多统计套利策略在流动性差的"配对"中表现最好（因为价差偏离更大，回归更确定）。但低流动性也意味着你进出困难——当你需要平仓时，可能没有对手盘。

一句话总结：**不要在流动性差的股票上做统计套利。** 如果日均成交额低于5000万，就不要碰。

## 一个完整的配对交易系统

说了这么多陷阱，不是说统计套利不可行——而是说它需要更严谨的设计。下面是一个可执行的系统框架：

![协整分析框架](/images/2026-06-01-stat-arbitrage-pairs-trading/cointegration.jpg)

**配对筛选阶段**：
- 在同行业内筛选候选配对（银行-银行、白酒-白酒），避免跨行业配对（基本面差异太大）
- 用滚动2年的数据进行ADF检验，p值<0.05才纳入
- 要求相关性>0.8作为辅助条件
- 检查对冲比率（β）是否稳定——如果β在最近半年剧烈波动，说明关系正在解体

**信号生成阶段**：
- 计算价差的z-score（标准化偏离）
- 开仓信号：|z-score| > 2.0
- 平仓信号：|z-score| < 0.5（回归到均值附近）
- 止损信号：|z-score| > 3.5（价差继续扩大，关系可能已经破裂）

**仓位管理**：
- 每个配对分配总资金的5-8%
- 多空两端按对冲比率配比（保证组合beta接近0）
- 同时持仓最多3-5个配对（避免集中风险）

## 统计套利在A股的适用性

坦诚地说，A股做统计套利有几个天然的障碍：

第一，**做空机制不完善**。融券标的有额度限制，很多中小盘股根本借不到券。这意味着在A股你只能做"半个配对交易"——做多低估的、但没法做空高估的。

第二，**T+1制度**。你今天看到信号买入，要明天才能卖出。一天的延迟对于价差交易来说可能是致命的。

第三，**散户结构**。A股的个人投资者占比高，非理性波动更大，价差偏离会更剧烈但回归也更不确定。

但这些障碍不意味着完全放弃。一个折中的思路是：**用ETF替代个股做配对交易**。行业ETF可以融券（部分标的），流动性好，而且不受单只股票基本面的干扰。

例如：银行ETF vs 房地产ETF、消费ETF vs 医药ETF——这些行业对之间存在此消彼长的关系，价差更加稳定。

## 最后的忠告

统计套利的魅力在于它在数学上是优美的——你不需要预测方向，只需要押注关系回归。但这也是它的危险所在：数学的优雅容易让人忽视现实世界的复杂性。

如果你要尝试统计套利，记住三条红线：
1. **永远设止损**——关系可能永久性断裂
2. **计入所有成本**——包括借券、滑点、冲击成本
3. **控制仓位**——任何一个配对不超过总资金的10%

统计套利不是一个"躺着赚钱"的策略。它是一个需要持续监控、严格执行、随时准备认错的系统。但如果你能做到这些，它确实提供了一种不依赖方向判断的收益来源——这在震荡市中尤其珍贵。
