# 多因子选股实战：从因子挖掘到组合构建

多因子选股是量化投资中最经典、最成熟的策略之一。从Fama-French三因子模型到如今的数百个因子，如何在海量因子中找到真正有效的那些？本文将带你从理论到实战，完整走一遍多因子选股流程。

## 什么是多因子模型？

多因子模型认为股票收益可以由多个风险因子解释。通过暴露在某些因子上，可以获得相应的风险溢价。

经典因子包括：
- **市值因子（Size）**：小盘股长期跑赢大盘股
- **价值因子（Value）**：低市盈率、市净率股票估值修复带来超额收益
- **动量因子（Momentum）**：过去表现好的股票继续表现好

## 因子挖掘：从直觉到数据验证

### 1. 因子来源

因子可以从三个维度挖掘：

**基本面因子**：
- 估值：PE、PB、PS、EV/EBITDA
- 盈利：ROE、ROA、毛利率
- 成长：营收增长率、净利润增长率

**技术面因子**：
- 动量：过去N天收益率、RSI、MACD
- 波动率：历史波动率、ATR
- 成交量：量价背离、资金流向

**另类因子**：
- 分析师预期调整
- 社交媒体情绪
- 供应链数据

### 2. 单因子测试

每个因子都要经过严格的IC分析（Information Coefficient）和分层回测：

```python
# 伪代码示例
factor_ic = spearman_corr(factor_value, next_20d_return)
if abs(factor_ic) > 0.02 and ic_t_stat > 2:
    print("因子有效！")
```

有效因子的标准：
- IC均值绝对值 > 0.02
- IC_IR（信息比率）> 0.3
- 分层回测单调性明显

## 因子组合：如何把多个因子结合？

### 方法一：等权加权
最简单粗暴，所有因子标准化后等权相加。

### 方法二：IC_IR加权
根据每个因子的预测能力动态加权，IC_IR高的因子权重更大。

### 方法三：机器学习模型
用XGBoost、LightGBM或神经网络学习因子与收益的非线性关系。

## 实战案例：构建中证500增强策略

### 数据准备
- 股票池：中证500成分股
- 数据频率：日频
- 回测区间：2018-01-01 至 2025-12-31

### 因子选择
经过测试，我们保留以下5个因子：
1. 20日动量（IC=0.04, IC_IR=0.45）
2. 市值（IC=-0.03, IC_IR=0.38）
3. ROE（IC=0.025, IC_IR=0.32）
4. 换手率（IC=-0.035, IC_IR=0.41）
5. 分析师评级变化（IC=0.03, IC_IR=0.35）

### 组合构建
每周调仓，选前30只股票等权持有，行业中性约束。

### 回测结果
- 年化收益：18.5%
- 基准年化：6.2%
- 超额收益：12.3%
- 最大回撤：-15.8%
- 信息比率：2.1

## 注意事项

1. **因子衰减**：有效的因子会随时间失效，需要持续挖掘新因子
2. **过拟合风险**：因子太多容易过拟合，建议控制在10个以内
3. **交易成本控制**：高频调仓会被交易成本吃掉收益，要估算滑点和手续费
4. **极端行情处理**：2024年2月微盘股暴跌，市值因子单月亏损20%+

多因子选股不是圣杯，但它是量化投资最扎实的基本功。关键在于持续迭代因子库、严格控制风险、保持对市场变化的敏感度。下期我们将讲解如何用机器学习提升因子合成效果。
