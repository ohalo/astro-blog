## 引言

配对交易（Pairs Trading）是最经典的统计套利策略之一，其核心理念是寻找两个历史价格走势高度相关的资产，当价差偏离常态时做多低估资产、做空高估资产，等待价差回归获利。本文将深入探讨配对交易的完整实务流程。

## 协整检验：配对交易的基石

### 为什么需要协整而非简单相关？

很多初学者误以为只要两只股票相关系数高就能配对，这是错误的。相关系数衡量的是**同期**线性关系，而协整性（Cointegration）衡量的是**长期均衡关系**。

**协整的关键特征：**
- 两只股票的线性组合是平稳过程
- 价差（Spread）具有均值回归特性
- 即使各自都是非平稳的时间序列（如随机游走），其线性组合也可以是平稳的

### Augmented Dickey-Fuller (ADF) 检验

实际应用中，我们采用 ADF 检验来验证价差是否平稳：

```python
from statsmodels.tsa.stattools import adfuller
import numpy as np

def check_cointegration(price1, price2):
    """检验两个价格序列的协整性"""
    # 线性回归求对冲比率
    X = price2.values.reshape(-1, 1)
    y = price1.values
    beta = np.linalg.lstsq(X, y, rcond=None)[0][0]
    
    # 计算价差
    spread = price1 - beta * price2
    
    # ADF检验
    adf_result = adfuller(spread, autolag='AIC')
    
    return {
        'adf_statistic': adf_result[0],
        'p_value': adf_result[1],
        'hedge_ratio': beta,
        'is_cointegrated': adf_result[1] < 0.05
    }
```

**判断标准：**
- p-value < 0.05：拒绝原假设（存在单位根），价差是平稳的，两资产协整
- p-value ≥ 0.05：不能拒绝原假设，价差非平稳，不适合配对

## 配对选择：如何筛选优质配对？

### 基本面筛选

协整性是必要条件，但非充分条件。优质的配对交易还需满足：

1. **同行业或相似业务模式**：确保长期均衡关系有经济逻辑支撑
2. **市值相近**：避免流动性差异过大导致的执行风险
3. **历史相关性 > 0.7**：保证价差的均值回归特性

### 技术指标筛选

除了协整检验，还可以考察：

- **半衰期（Half-life）**：价差回归均值所需时间的对数尺度
  - 计算公式：HL = ln(2) / |λ|，其中 λ 是 Ornstein-Uhlenbeck 过程的均值回归速度
  - 实务中，半衰期在 5-20 个交易日较为理想
  
- **Hurst 指数**：衡量时间序列的记忆性
  - H < 0.5：均值回归（适合配对交易）
  - H = 0.5：随机游走
  - H > 0.5：趋势性

## 交易信号设计

### Z-Score 标准化

将价差标准化为 Z-Score，便于设定统一的入场和出场阈值：

```python
def calculate_z_score(spread, window=20):
    """计算价差的Z-Score"""
    mean = spread.rolling(window=window).mean()
    std = spread.rolling(window=window).std()
    z_score = (spread - mean) / std
    return z_score
```

### 入场与出场阈值

**经典设置：**
- 入场：|Z-Score| > 2.0（做多低估、做空高估）
- 出场：Z-Score 回归到 0 附近（如 |Z-Score| < 0.5）
- 止损：|Z-Score| > 3.0（价差持续扩大，协整关系可能失效）

**动态阈值优化：**
可以采用分位数法动态调整阈值，适应不同市场环境下的波动率变化。

## 实盘执行要点

### 1. 对冲比率的动态更新

协整关系并非永久不变，建议：
- 每月重新估计对冲比率（β）
- 当 β 变化超过 10% 时，考虑平仓并重新建仓
- 采用 Kalman Filter 实时更新对冲比率（进阶方法）

### 2. 交易成本控制

配对交易属于高频策略，交易成本影响巨大：

- **双边换手率**：每次交易需同时买入和卖出，交易成本翻倍
- **滑点管理**：使用限价单（Limit Order）避免市价单的大滑点
- **冲击成本**：大单拆分执行，避免对市场造成冲击

### 3. 风险管理

**仓位管理：**
- 单对配对的最大仓位不超过总资金的 5%
- 同时持有 10-20 对配对，分散非系统性风险

**止损规则：**
- 时间止损：持仓超过 30 个交易日仍未收敛，强制平仓
- 价值止损：价差的 Z-Score 突破 -3.5 或 +3.5

## 中国市场的实务考量

### 做空限制

A 股市场做空机制有限，可采用以下替代方案：

1. **融券卖出**：部分标的可融券，但成本和可得性受限
2. **股指期货对冲**：用沪深 300/中证 500 期货近似替代做空
3. **反向 ETF**：部分市场有杠杆反向 ETF，但跟踪误差较大

### 涨停板限制

中国市场的涨跌停板制度可能导致：
- 无法及时平仓
- 价差持续扩大而无法收敛

**应对策略：**
- 避开波动率极高的时期（如政策发布日）
- 设置更严格的止损阈值

## 绩效评估

一个典型的配对交易策略应关注以下指标：

| 指标 | 优秀水平 | 合格水平 |
|------|---------|---------|
| 年化收益率 | > 15% | > 8% |
| 夏普比率 | > 2.0 | > 1.5 |
| 最大回撤 | < 5% | < 10% |
| 胜率 | > 60% | > 50% |
| 平均持仓周期 | 5-15 天 | 3-30 天 |

## 结语

配对交易看似简单，实则需要严谨的统计检验、精细的信号设计和严格的实盘执行。成功的配对交易员往往是：
- **统计学家**：深入理解协整、平稳性等概念
- **程序员**：能够高效回测和优化策略
- **交易员**：懂得风险管理和执行技巧

在量化交易竞争日益激烈的今天，配对交易仍是值得深入研究的策略。关键在于**持续挖掘新的配对机会**、**优化执行算法**，以及**严格控制风险**。
