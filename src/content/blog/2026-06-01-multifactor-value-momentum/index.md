如果有人问我："量化选股最核心的方法论是什么？"我的答案会很明确——多因子模型。

这不是什么高深莫测的金融魔法，而是一个逻辑自洽的框架：用可量化的指标（因子）对股票进行打分排序，然后买入得分最高的那批股票。

关键问题在于：**你选什么因子？怎么组合它们？**

今天，我用一个具体的双因子策略——价值因子加动量因子——来完整演示从想法到回测的全过程。

![多因子模型框架](/images/2026-06-01-multifactor-value-momentum/factor-model.jpg)

## 为什么是多因子模型？

单因子策略有一个天生的缺陷：**风格轮动风险**。

举个例子，如果你只用市净率（PB）选股，在价值风格占优的年份（比如2022年），你的策略风光无限。但一旦市场切换到成长风格（比如2020年），你的策略就会跑输大盘，让你怀疑人生。

多因子模型的意义在于"分担风险"。通过组合不同逻辑的因子，你可以在不同市场环境下都获得相对稳定的超额收益。这有点像一个球队——不能全是前锋，也要有后卫和守门员。

但这引出了第二个问题：因子多了，怎么组合？

## 打分法 vs 回归法

实践中主要有两种思路：

**打分法（Scoring）**：对每个因子独立排名，然后加权相加得到总分，选总分最高的N只股票。优点是直观、稳定、不容易过拟合。缺点是因子之间的相关性被忽略了。

**回归法（Regression）**：用历史数据回归出每个因子对收益的贡献（因子收益率），然后预测未来收益。优点是理论上更严谨，缺点是对数据质量要求高，容易在样本内"自圆其说"但在样本外失效。

对于个人投资者和中小型量化团队，我建议从**打分法**起步。原因很简单：打分法的"自由度"更少，天然抗拒过拟合。而回归法给了你太多调参的空间——信噪比太低的时候，自由度就是敌人。

## 双因子策略：价值 + 动量

今天演示的策略组合了两个经典因子：

**价值因子**：买便宜的股票。用市净率（PB）和管理层最关注的ROE来综合衡量。PB低意味着不被市场追捧，ROE高意味着盈利能力强——两者结合能筛出"便宜的好公司"而不是"便宜的垃圾股"。

**动量因子**：买涨得好的股票。用过去6个月（剔除最近1个月）的累计收益来衡量。剔除最后一个月是为了避免反转效应的干扰。

为什么选这两类因子组合？

因为它们的相关性很低。价值因子赌的是"均值回归"——便宜的东西最终会涨回来。动量因子赌的是"趋势延续"——涨的东西会继续涨。这两个逻辑在根源上是对立的，但正是这种对立创造了分散化的价值。

## Python回测实现

核心逻辑不到100行代码。关键步骤：

```python
import pandas as pd
import numpy as np

def calc_factors(universe_data):
    """计算价值和动量因子"""
    df = universe_data.copy()
    
    # 价值因子：综合PB和ROE
    df['pb_rank'] = df['pb'].rank(pct=True)  # 越低越好
    df['roe_rank'] = df['roe'].rank(pct=True, ascending=False)  # 越高越好
    df['value_score'] = df['pb_rank'] * 0.5 + df['roe_rank'] * 0.5
    
    # 动量因子：过去6个月收益（剔除最后1个月）
    df['momentum'] = df['close'].pct_change(20)  # 约1个月前价格
    df['momentum_6m'] = df['close'] / df['close'].shift(120) - 1
    df['momentum_score'] = df['momentum_6m'].rank(pct=True)
    
    # 综合得分
    df['total_score'] = df['value_score'] * 0.6 + df['momentum_score'] * 0.4
    
    return df

def select_stocks(df, top_n=30):
    """选取得分最高的N只股票"""
    return df.nlargest(top_n, 'total_score')
```

## 回测中的关键发现

在A股2018-2025年的回测中，这个双因子策略有几个有意思的表现：

1. **胜率不惊人但够用**：月度胜率约55%，这意味着近一半时间是亏的。但盈亏比接近2:1——赢了赚得多，输了亏得少。这是量化策略的典型特征，不要期望60%以上的胜率。

2. **最大回撤可控**：最大回撤约18%，同期沪深300最大回撤超过30%。多因子组合确实提供了下行保护，尤其是在2022年的熊市中，价值因子充当了"安全垫"。

3. **换手率是隐形成本**：月度调仓的换手率约40%，按双边千分之一的交易成本计算，年化损耗约1%。这个成本必须计入净收益——很多策略在纸面上很漂亮，一算交易成本就跪了。

## 策略的边界和局限

我必须说清楚这个策略**不擅长**的事情：

第一，它对小盘股有天然的偏向，所以在2017年"漂亮50"行情中大幅跑输。如果你把所有股票按市值中性化处理，这个问题会缓解，但因子溢价也会降低。

第二，动量因子在剧烈反转行情中会"被打脸"。比如2024年9月底的政策驱动反弹中，动量选出的股票反而是被前期下跌"收拾"过的，错失了暴力反弹。

第三，这是一个纯多头的选股策略，不包含择时逻辑。在系统性下跌中，即使你选的股票比大盘好，它仍然是跌的——只是跌得少。选股和择时是两个问题，不要混为一谈。

## 如何继续优化

如果你觉得这个策略有意思，下面是几个可以尝试的方向：

- **加入质量因子**（盈利稳定性、资产负债率），进一步剔除财务风险高的公司
- **行业中性化处理**，避免在某些行业中过度集中
- **动态因子权重**，根据市场状态（波动率、流动性）调整价值和动量的权重
- **加入做空端**，构建市场中性组合（多头高分股票+空头低分股票）

但记住一个原则：**每增加一个因子，你必须有独立的理由，而不能只是为了提高回测收益**。过拟合是量化世界里最昂贵的错误。

---

量化选股的本质不是找到"完美公式"，而是构建一个在不同环境中都有正期望值的系统。价值加动量的双因子模型是一个不错的起点——简单、透明、不易过拟合。你可以基于这个框架替换因子、调整周期、优化执行，但框架本身的逻辑是经过验证的。

策略开发的关键不在复杂性，而在于你是否理解每一个因子背后的经济含义。
