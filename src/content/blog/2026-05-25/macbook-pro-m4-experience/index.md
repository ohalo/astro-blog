用了三年的 Intel MacBook Pro 在某次编译时键盘发烫到可以煎鸡蛋之后，我终于下定决心换机了。选的是 14 英寸 MacBook Pro，搭载 M4 芯片，24GB 统一内存，1TB 存储。到今天已经使用了三个月，有些体验想分享出来。

## 为什么选 M4 而不是等待 M5

说实话，选 M4 是「务实」战胜「追新」的结果。M5 芯片的传言从去年底就开始了，但按照苹果的节奏，M5 MacBook 最早也要今年秋季才能见到，而且初期供货紧张、价格偏高。

M4 本身已经是第四代自研芯片，经过 M1、M2、M3 的迭代，生态适配成熟，软件对 Apple Silicon 的优化已经相当完善。三个月用下来，我可以负责任地说：**M4 不是过渡产品，而是一个真正的成熟节点**。

## 性能：编译速度的飞跃

我最核心的工作负载是 TypeScript 项目的前端开发和偶尔的 Python 数据处理。M4 给我最直观的感受是编译和构建速度的提升。

之前的 Intel 16 英寸 MacBook Pro（i9，32GB 内存），跑一个中等规模的前端项目 `npm run build`，需要大约 45 秒。换成 M4 之后，同样的项目只需要 18 秒。这个差距在日常开发中会不断累积——每天如果构建 20 次，节省下来的时间就相当可观了。

更重要的是发热和噪音控制。Intel 时期的 MacBook 在编译时风扇会「起飞」，键盘区域明显发烫。M4 的发热量极低，日常开发中几乎感觉不到机器的温度，更谈不上风扇噪音。这是一个质变，不是量变。

统一内存架构的优势在这三个月里也体现得很明显。之前跑一个数据处理脚本，同时开着 Chrome（十几个标签页）、VS Code、Slack、Notion，内存使用率经常飙到 90% 以上，系统开始 swap。M4 这台机器同样负载下，24GB 内存绰绰有余，内存压力始终是绿的。

## 屏幕和音响：依然顶级

MacBook Pro 的屏幕一直是行业标杆，M4 版本也不例外。Liquid Retina XDR 显示屏的 HDR 显示效果在看视频和处理图片时依然是享受。120Hz ProMotion 自适应刷新率让滚动和动画丝滑流畅，用惯了之后再去看 60Hz 的屏幕会明显感觉「卡」。

不过有一点我想吐槽：苹果坚持刘海屏设计已经到第四代了。客观说，这个刘海在日常使用中确实不影响体验，但看到它还是会让人怀念早期没有刘海的全面屏设计。这是一个「功能性」和「美观性」的权衡，苹果选择了前者。

音响系统依然是笔记本中的天花板。六个扬声器组成的音响系统配合空间音频支持，在看剧和听音乐时的表现足以替代大多数桌面音箱。当然，如果你是严肃的音乐制作还是需要专业的监听设备，但日常娱乐绰绰有余。

## 接口扩展性：终于够用了

这一代 MacBook Pro 配备了三个雷雳 4 接口、一个 HDMI 接口、一个 SD 卡槽和一个 MagSafe 3 充电接口。相比之前只有雷雳接口的时代，这个配置让我的日常使用方便了很多。

HDMI 接口让我在需要演示时可以直接连投影仪，不需要每次找转接头。SD 卡槽对经常需要导照片的我来说是刚需，拍完照直接插卡导入，比读卡器方便多了。

MagSafe 3 磁吸充电也值得一说。它不仅充电方便（轻轻一放就吸上），更重要的是安全性——有人绊到线缆时，磁吸头会断开而不是把机器拽到地上。这个设计在办公室场景中特别实用。

## 续航：真实使用感受

官方宣传的续航数字是「最长 24 小时」，实际使用中我从来没有跑到过这个数字——当然，这是可以预期的，因为真实使用场景比实验室测试复杂得多。

我的实际体验是：正常工作日（写代码、开浏览器、开通讯软件），从早上 9 点用到下午 6 点，电量从 100% 掉到约 30%。这意味着如果不带充电器，中等强度使用一天是勉强可以的，但低电量焦虑会比较明显。

但 M4 的低功耗表现有一个让我惊喜的点：待机功耗极低。周五下班合上盖子，周一上班打开盖子，电量几乎不掉。这个体验和 iPad 很像，让我不再有「下班要不要关机」的纠结。

## 三个月使用后的遗憾

说了这么多优点，也要谈谈不满足的地方。

首先是价格。MacBook Pro M4 的配置价格对于普通消费者来说依然偏高。我的配置（24GB + 1TB）加上 AppleCare+，总花费接近两万五千元。这个价格能买到一台相当不错的 Windows 创作本或者游戏本了。

其次是苹果的存储定价策略。升级到 1TB 需要额外付 1500 元，这个溢价依然偏高。苹果在存储上的利润率一直是果粉心中的痛。

第三，macOS 在某些专业软件支持上仍有短板。虽然 Adobe 全家桶和大多数开发工具都已经原生支持 Apple Silicon，但某些小众工具或行业专用软件还停留在 Rosetta 2 转译阶段，性能有损失。

## 适合人群分析

如果你正在使用 M1 或更早的 MacBook，升级到 M4 的体验会非常明显，性能、续航、发热控制都有质变。但如果你是 M3 用户，M4 的提升更多是量变而非质变，升级必要性不大。

如果你正在从 Intel Mac 切换，M4 绝对是值得的——统一内存架构带来的改变是颠覆性的，你会发现用了三年的机器像是上个时代的产品。

如果你主要用 Windows，或者你的工作依赖某些只在 Windows 上运行的软件，MacBook Pro 再好也不适合你。工具要选对的，不是选贵的。

## 写在最后

三个月的深度使用之后，我对 MacBook Pro M4 的评价是：**这是一台能够「让你忘记硬件」的生产力工具**。它的存在感很低——没有噪音、没有烫手、没有续航焦虑——但它的产出能力很强。这种「隐身」的感觉，恰恰是我对生产力工具的最高评价。

当你不再需要担心「机器能不能跑动这个项目」，不再需要在性能和噪音之间权衡，不再需要带着充电器出门的时候，你就知道，这代芯片，终于毕业了。
