MACD（Moving Average Convergence Divergence）是技术派最常被挂在嘴边的指标，但大多人只背「金叉买、死叉卖」，却说不清金叉到底在哪条线、红绿柱凭什么代表动能。本文先把公式拆干净，再用**无前瞻回测**验证一个常被掩盖的真相：MACD 几乎不创造超额收益，它的真价值是**把崩盘段的回撤砍掉一半**——一把风控尺，不是印钞机。

## 一、四个量：从两条 EMA 差分而来

MACD 的全部信息都在两条指数移动平均(EMA)的差里。设快线跨度 `FAST=12`、慢线 `SLOW=26`、信号线 `SIG=9`：

```
EMA_t(span) = α·x_t + (1−α)·EMA_{t-1}(span),   α = 2/(span+1)
DIF   = EMA(12) − EMA(26)            # 快减慢, 动能差
DEA   = EMA(DIF, 9)                  # DIF 的 9 期平滑, 信号线
MACD柱 = 2 · (DIF − DEA)             # 柱越长, 快慢线开口越大
```

直觉：价格加速上涨 → 快线冲在慢线上方 → DIF 为正且扩大 → 红柱伸长（动能增强）；涨势衰竭 → 快线回落贴近慢线 → DIF 收敛 → 红柱缩短；转跌 → DIF 下穿 DEA（死叉）→ 绿柱出现。

```python
import numpy as np

def ema(x, span):
    a = 2.0 / (span + 1)
    out = np.empty_like(x, dtype=float)
    out[0] = x[0]
    for i in range(1, len(x)):
        out[i] = a * x[i] + (1 - a) * out[i - 1]
    return out

FAST, SLOW, SIG = 12, 26, 9
ema_f = ema(price, FAST)
ema_s = ema(price, SLOW)
dif   = ema_f - ema_s
dea   = ema(dif, SIG)
hist  = 2 * (dif - dea)
```

## 二、DIF / DEA / 红绿柱：动能拐点的经典图

把 DIF、DEA 和红绿柱叠在一起，就是交易软件里最经典的 MACD 面板。注意**金叉= DIF 上穿 DEA（绿柱翻红），死叉= DIF 下穿 DEA（红柱翻绿）**——所有信号都源自这两条线的相对位置，没有任何神秘：

## 三、回测：信号次日开盘交易，杜绝前瞻偏差

最常见的 MACD 回测造假，是「今日出金叉、今日收盘价买入」——收盘才能算 DIF/DEA，收盘价买入等于用了收盘后的信息。正确做法是：**用 t-1 收盘确认的信号，决定 t 开盘的持仓**，赚取 `[t, t+1]` 这段收益。

```python
cross_up = (dif > dea) & (np.r_[False, dif[:-1] <= dea[:-1]])   # 金叉
cross_dn = (dif < dea) & (np.r_[False, dif[:-1] >= dea[:-1]])   # 死叉
idx_up, idx_dn = np.where(cross_up)[0], np.where(cross_dn)[0]

pos = np.zeros(N, dtype=int); state = 0
for t in range(1, N):
    if t in idx_up: state = 1        # 金叉 -> 持有
    elif t in idx_dn: state = 0      # 死叉 -> 空仓(转现金)
    pos[t] = state

# 关键: 用 t-1 信号(pos[:-1])吃 t 的收益, 避免同日信号同日执行的前瞻偏差
strat_ret = np.r_[0.0, pos[:-1] * (price[1:] / price[:-1] - 1)]
bh_ret    = np.r_[0.0, price[1:] / price[:-1] - 1]
strat_nav = np.cumprod(1 + strat_ret)
bh_nav    = np.cumprod(1 + bh_ret)
```

## 四、诚实结果：不创造 alpha，但砍掉一半回撤

我们用合成价格（分段制度：上涨段 / 横盘高噪声段 / 下跌段交替），确保趋势与震荡都充分出现。1600 个交易日、62 次金叉、63 次死叉，无交易成本：

| 策略 | 年化 | 波动 | 夏普 | 最大回撤 |
|---|---|---|---|---|
| MACD 策略 | 13.71% | 9.03% | **1.30** | **−17.74%** |
| 买入持有 | 17.57% | 12.53% | 1.24 | −35.31% |

结论可能会让技术派不舒服：**MACD 没有跑赢买入持有**（13.7% < 17.6%），夏普也只是略高（1.30 vs 1.24）。原因很直白——双 EMA 都滞后，趋势段里它总是「晚半拍进场、晚半拍离场」，把最肥的一段让给了 buy&hold。

但它做对了一件事：死叉后转到现金，**躲过了下跌段**。最大回撤从 −35% 砍到 −18%，几乎减半。这才是 MACD 的真实定位——**崩盘保护器**，不是收益放大器：

## 五、假信号指纹：横盘段鞭锯率更高

MACD 在教科书里的头号警告是「横盘市反复假叉、来回止损」。我们用「金叉后 15 日内即出现死叉」定义一次鞭锯（whipsaw），统计两类市场：

- 上涨趋势段金叉：鞭锯率 **57.5%**(n=40)
- 横盘段金叉：鞭锯率 **61.5%**(n=13)

横盘段鞭锯率确实更高——趋势段金叉后价格多能延续，横盘段价格原地打转，金叉死叉高频交替，把利润吐回手续费。这是 MACD 的结构性弱点，任何「MACD 稳赚」的说法都该先过这一关：

## 六、四类致命陷阱（务必先看这段）

1. **滞后性（最致命）**：EMA(12) 和 EMA(26) 都是对过去的加权平均，信号天然晚于价格拐点。趋势越猛，你进场越晚、离场越晚——这是数学决定的，换参数只能缓解不能消除。
2. **横盘鞭锯**：如上图，震荡市里金叉死叉密集交替，反复「买入→止损→买入」，磨损极快。实务必须叠加波动率过滤或趋势强度过滤（如 ADX）才敢用。
3. **参数敏感**：12/26/9 是 1970 年代美股日线的经验值。换到周线、加密币、或 A 股，最优参数天差地别；更糟的是**最优参数几乎一定过拟合**——样本内漂亮、样本外崩。
4. **前瞻偏差陷阱**：任何「当日金叉当日收盘买入」的回测都虚高了收益。本文用 `pos[:-1]` 吃 `t` 收益，是消除该偏差的标准写法；若看到某 MACD 策略回测夏普 >2，先查它有没有偷看收盘。

## 七、正确用法：当「开关」，别当「引擎」

把 MACD 放回它该在的位置：

- **趋势过滤器**：只在 DIF > 0（多头动能）时持有权益，DIF < 0 时降仓——本文化身就是「死叉转现金」；
- **仓位旋钮**：用 MACD 柱高度估计动能强度，柱越长敞口越高，而非简单 0/1；
- **配对使用**：单独 MACD 不够，必须配波动率过滤（横盘期关掉）或更长周期确认（周线金叉才信）。

## 结语

MACD 的全部数学，就是「快慢两条 EMA 的差」再平滑一次。它不神秘、不印钞——在诚实的无前瞻回测里，它甚至跑不赢买入持有。但它做对了一件值钱的事：**在动能转负时退到现金，把 −35% 的回撤砍成 −18%**。把它当风控开关、当趋势过滤器，它物有所值；把它当 alpha 引擎、当圣杯，市场会教你怎么亏钱。理解它的滞后与鞭锯，比记住「金叉买」重要一百倍。
