引言：量化交易是什么？为什么要学？

🤔 想象一下

你是一个基金经理，每天要面对海量的市场数据：股价、成交量、财报、新闻... 靠人工分析，根本看不过来。更糟的是，人性是情绪化的——恐惧、贪婪、侥幸心理，往往导致追涨杀跌、割肉在地板。

💡 量化交易就是答案

用量化的方法（数学、统计、编程）来研究市场、制定策略、自动执行交易。它不靠感觉，不靠小道消息，只相信数据和回测结果。

📈 量化的核心优势

  • 纪律性：严格执行策略，不受情绪干扰
  • 系统性：可以同时监控上千只股票，发现人工看不到的规律
  • 可验证：用历史数据回测，策略好不好，数据说了算
  • 可扩展：策略写好后，可以7×24小时自动运行

import pandas as pd
import numpy as np

# 计算价值因子（PB比率）
def calculate_value_factor(df):
    df['value_factor'] = 1 / df['pb_ratio']
    return df

# 计算动量因子（过去12个月收益率，排除最近1个月）
def calculate_momentum_factor(df):
    df['momentum_factor'] = df['return_12m'] - df['return_1m']
    return df
